# Autopsy Master

**From scene → table → truth.**

A comprehensive, mobile-first, dark-mode, offline-ready educational platform for autopsy and postmortem examination training.

## Overview

Autopsy Master is designed for pathology residents, autopsy fellows, and forensic trainees. The platform covers the complete workflow of hospital and forensic autopsy examination, from body intake to final report.

## Features

- **12 Comprehensive Modules** covering all aspects of autopsy practice
  - Introduction & Safety
  - External Examination
  - Internal Examination Techniques
  - Natural Disease Pathology
  - Forensic Pathology Foundations
  - Postmortem Changes & Artifacts
  - Toxicology & Ancillary Testing
  - Histology in Autopsy
  - Hospital Autopsy Workflow
  - Forensic Case Workflow
  - Cause-of-Death Reasoning
  - Integrated Autopsy Cases

- **Interactive Tools** for hands-on practice
  - Organ Weight Analyzer
  - Cause-of-Death Mapper
  - External Exam Builder
  - Injury Classifier
  - And more...

- **Session-Based Assessments** to test knowledge (no tracking)
- **Mobile-First Design** optimized for all devices
- **Offline Support** via Progressive Web App (PWA)
- **Dark Mode** default for reduced eye strain

## Technology Stack

- **React** + **TypeScript** for type-safe component development
- **Vite** for fast builds and development
- **Tailwind CSS** for utility-first styling
- **React Router** for client-side routing
- **Zustand** for session-only state management
- **Lucide React** for beautiful icons

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Production Build

```bash
npm run build
```

The built files will be in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

## Educational Philosophy

This platform emphasizes:

- **Pattern Recognition** across natural disease, trauma, and artifacts
- **Systematic Workflow** from external to internal examination
- **Safety Protocols** and biosafety fundamentals
- **Clinical Correlation** between gross and microscopic findings
- **Cause-of-Death Reasoning** using logical causal chains
